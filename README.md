# Set up guide

## Prerequisites
- node

Download from <https://nodejs.org/en/download/>

## How to run
Using your preferred commandline editor
```shell
$ cd yourFolder
$ git clone https://github.com/inspiraller/markdown_to_html_demo.git
$ cd markdown_to_html_demo
$ npm install
$ npm start
```
